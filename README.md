# Fitelo Website Manual Testing Project

## Project Overview

This project demonstrates a beginner level manual testing practice
performed on the Fitelo website. The goal of this project is to
understand how a QA tester analyzes a real product, designs test cases,
executes testing scenarios, and reports bugs.

## Website Tested

https://fitelo.co

## Testing Scope

The testing focused on validating the main user experience of the
website including navigation, page loading, form validation, and
responsiveness.

## Testing Activities

Test plan preparation Test case design Manual execution of test cases
Bug identification and documentation

## Test Environment

Browser: Google Chrome Operating System: Windows

## Project Deliverables

Test Plan document Test Cases spreadsheet Bug Report document

## Skills Practiced

Manual Testing Test Case Design Bug Reporting Website Functional Testing
UI Validation

## Learning Outcome

Through this project I practiced how to approach a real website from a
tester perspective, identify possible issues, and document testing
activities in a structured and professional way.
